# Svelte + Tailwind

## How to use

- Click `Use this template` => `Create a new repository`.
- Provide your new repository informations.
- Clone your repository, open it :
```shell
mkdir <your_repository>
git clone <link>
cd <your_repository>
code .
```
- Init your brand new **Svelte + Tailwind**, start it :
```shell
npm i
npm run dev
```
- Open `localhost:5173/`
